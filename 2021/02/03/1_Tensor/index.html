第一章 预备知识-Tensor

1.1数据操作

在Pytorch中，torch.Tensor是一种单一数据类型的多维矩阵，又名张量

Tensor与Numpy的多维数组非常类似，但是作为专门为DL设计，Tensor支持GPU计算和自动求导等功能

1.1.3 广播机制

广播机制在python中也非常常见，相比传统C/C++等语言，广播机制加速了矩阵的运算

当对两个形状不一致的数组进行运算时，会触发广播机制：先适当复制元素使得两个数组同形状再接着运算

(tensor([[0],
         [1],
         [2]]),
 tensor([[1, 2]]))
tensor([[1, 2],
        [2, 3],
        [3, 4]])

1.2 自动求梯度

在DL中经常要用到求梯度(gradient)，Pytroch提供了autograd模块来自动求梯度

以函数y=2x^2为例

requires_grad与backward的使用

requires_grad=True之后开始追踪X的所有操作，也即将函数表达式记录下来

1.3.2 查找特定函数和类的使用：help函数

Help on built-in function zeros_like:

zeros_like(...)
    zeros_like(input, dtype=None, layout=None, device=None, requires_grad=False) -> Tensor

    Returns a tensor filled with the scalar value `0`, with the same size as
    :attr:`input`. ``torch.zeros_like(input)`` is equivalent to
    ``torch.zeros(input.size(), dtype=input.dtype, layout=input.layout, device=input.device)``.

    .. warning::
        As of 0.4, this function does not support an :attr:`out` keyword. As an alternative,
        the old ``torch.zeros_like(input, out=output)`` is equivalent to
        ``torch.zeros(input.size(), out=output)``.

    Args:
        input (Tensor): the size of :attr:`input` will determine size of the output tensor.
        dtype (:class:`torch.dtype`, optional): the desired data type of returned Tensor.
            Default: if ``None``, defaults to the dtype of :attr:`input`.
        layout (:class:`torch.layout`, optional): the desired layout of returned tensor.
            Default: if ``None``, defaults to the layout of :attr:`input`.
        device (:class:`torch.device`, optional): the desired device of returned tensor.
            Default: if ``None``, defaults to the device of :attr:`input`.
        requires_grad (bool, optional): If autograd should record operations on the
            returned tensor. Default: ``False``.

    Example::

        >>> input = torch.empty(2, 3)
        >>> torch.zeros_like(input)
        tensor([[ 0.,  0.,  0.],
                [ 0.,  0.,  0.]])
