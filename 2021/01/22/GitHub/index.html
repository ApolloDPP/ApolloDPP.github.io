利用git将整个项目代码文件上传到GitHub

注：GitHub不支持大于100M的文件上传

git代码

1.在GitHub上建立一个新仓库

2.git进入本地项目文件根目录

（git安装：https://git-scm.com/downloads/）

3.git init

会出现.git文件，表示本项目已经git初始化完成

4.git add .

注：后面的”.”要空格

表示将项目中的所有文件添加到仓库中

5.git commit -m “first commit”

表示你对这次提交的注释，双引号里面的内容可以根据个人的需要

6.git remote add origin +GitHub仓库的URL地址

7.git push -u origin master

将项目上传到GitHub仓库的最后操作
